# Post-MVP Roadmap

| Phase | 内容 | 完了条件 |
| --- | --- | --- |
| Alpha feedback | ユーザー手動テスト結果を収集 | manual-test の未実施項目が解消 |
| Data adapters | 実ファイル、ホストアプリ、外部ログの読み込みを追加 | 代表サンプル以外で pass/warn/fail が再現 |
| UI hardening | 操作導線、エラー表示、結果比較を改善 | 手動確認者が docs なしでも主要操作可能 |
| Distribution | BOOTH / Store / Release 向け素材を整備 | 配布ページ、説明画像、FAQ が揃う |
